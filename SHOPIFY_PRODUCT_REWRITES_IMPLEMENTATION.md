## 🚀 STEP-BY-STEP IMPLEMENTATION

### Step 1: Access Shopify Admin
1. Go to https://www.shopify.com/admin
2. Log in with your Pure Leven credentials
3. Click "Products" in the left sidebar
4. Search for the product (e.g., "Black Pepper")

### Step 2: Prepare the New Description
1. Have this file open with the product descriptions
2. For the specific product, find the section marked "NEW DESCRIPTION"
3. Copy everything between the opening `<h2>` and closing `</h2>` of the last section
4. Include all HTML formatting (headers, lists, paragraphs)

### Step 3: Replace in Shopify
1. In the product page, find the "Description" field
2. Click in the field
3. Select ALL text (Cmd+A on Mac, Ctrl+A on Windows)
4. Delete the old description
5. Paste the new HTML description
6. The Shopify editor will automatically format it

### Step 4: Preview & Test
1. Click "Preview" button (top right) or look at preview panel
2. Check on desktop view:
   - Are headers displaying correctly?
   - Are bullet points formatted?
   - Is text readable?
   - Do images show (if any)?
3. Check on mobile view (important!):
   - Is text readable on small screen?
   - Are lists formatted correctly?
   - Is spacing good?

### Step 5: Save & Publish
1. Click "Save" button (top right)
2. Product is now saved as draft
3. Click "Publish" to make live on store
4. You'll see "Published" confirmation

### Step 6: Verify Live
1. Go to https://www.pureleven.com/products/[product-slug]
2. Scroll through description
3. Check formatting on mobile (use browser dev tools or mobile phone)
4. Verify CTAs are visible

## ⚠️ COMMON ISSUES & FIXES

### Issue: Text looks weird/broken
**Fix**: Make sure you copied the ENTIRE description including all HTML tags. Check for any missing `<p>`, `<ul>`, `</li>` tags.

### Issue: Formatting doesn't look right on mobile
**Fix**: This is normal for complex descriptions. Shopify adjusts automatically. Preview in mobile view to verify readability.

### Issue: CTA buttons don't look right
**Fix**: The CTA text (`ADD TO CART`) will use Shopify's theme styling. It should automatically match your store's button style.

### Issue: Images aren't showing
**Fix**: These descriptions don't include images in the HTML. If you want images, add them via Shopify's media uploader separately.

### Issue: Description is too long
**Fix**: These descriptions are intentionally comprehensive. Long descriptions improve SEO and conversion. Keep them as-is.
