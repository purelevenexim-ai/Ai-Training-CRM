## 📝 IMPORTANT NOTES

### Bidding Strategy - Why Manual CPC is Best
- **Current:** Maximize Conversions (requires data to work well)
- **Problem:** With 0 conversions, algorithm has nothing to learn from
- **Solution:** Manual CPC gives you complete control
- **Future:** After 30+ conversions, switch back to Maximize Conversions for optimization

### Negative Keywords - How They Work
- **Broad Match (e.g., "tips"):** Blocks "spice tips", "cooking tips", "seasoning tips", "recipe tips"
- **Phrase Match (e.g., "pureleven sale"):** Blocks exact phrase but allows variations
- **Exact Match (e.g., "MDH"):** Only blocks exact brand names

### Search Terms Report - Regular Maintenance
- Check weekly for the first month
- Add 10-20 new negative keywords based on actual search data
- Remove negative keywords if they're blocking converting searches (rare but possible)
