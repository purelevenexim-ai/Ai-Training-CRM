**Q: What do I do first?**

**Q: How long until I see results?**
A: First conversions: Day 3-5. Clear ROAS: Week 2. Optimization: Month 2+

**Q: Do I need to create all 9 audiences?**
A: No. Start with 3 (Checkout, Cart, Non-Converters). Add others as you grow.

**Q: Should I run Google Ads system AND Meta system?**
A: YES. They're different traffic sources. Running both is 2-3x more powerful.

**Q: What if I don't see results in Week 1?**
A: Normal. Audience needs to build (24-48 hrs). Meta needs time to optimize. Give it 7+ days.

**Q: Which campaign should I launch first?**
A: Checkout Recovery. Highest ROI (5-10:1 expected). Get quick wins.

**Q: How much should I spend?**
A: Start conservative: ₹500-1,000/week. Scale if ROAS > 1.5:1

**Q: What if ROAS is low?**
A: Test different creatives. Ensure audience is right. Check tracking. Wait longer.

**Q: Can I use the same creatives as Google system?**
A: No. Different audiences need different messaging. Google = urgent. Meta = lifestyle.
