## 📞 Support

If templates fail to create or messages don't send:

1. Check Wabis dashboard for template approval status
2. Verify template names match exactly (case-sensitive)
3. Ensure parameter counts match: `{{1}}`, `{{2}}`, etc.
4. Check `/api/test/journey/log` endpoint for detailed error logs
5. Run with `--no-wait` flag to send all immediately and test faster

## 💡 Pro Tips

- **Test Approval**: Templates can be PENDING for testing; Meta approves within 1-2 hours
- **Quick Testing**: Run with `--no-wait` to send all 15 immediately instead of waiting 5 hours
- **Custom Intervals**: Use `--interval 300` for 5-minute delays instead of 20-minute delays
- **Monitor Live**: Keep `/api/test/journey/log` open to watch messages being sent in real-time
