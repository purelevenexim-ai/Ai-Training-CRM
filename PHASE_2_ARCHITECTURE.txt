QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════════════

Environment Variable:
  REACT_APP_CRM_API=https://track.pureleven.com/api/crm

Install:
  npm install zustand recharts

Import Main Dashboard:
  import CRMDashboard from './CRMDashboard_V2';

Three Main Views:
  1. Analytics Dashboard  → Real-time journey metrics
  2. Customer Timeline    → Individual customer journey history
  3. Journey Builder      → Visual drag-drop editor for new journeys

Starting Point:
  npm run dev
  Navigate to http://localhost:5173 (or your dev server)

Test Backend Health:
  curl https://track.pureleven.com/api/crm/health
