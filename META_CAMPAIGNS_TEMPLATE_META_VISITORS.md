## KEY DIFFERENCES: Meta Visitor Retargeting vs Google Ads Visitor

```
Google Ads Visitors:
├─ High intent (searching for solutions)
├─ Faster conversion (30-40% in first visit)
├─ Message: Problem → Solution → Buy
├─ ROAS: 3-8:1 (highest)

Meta Ads Visitors:
├─ Interest-based (awareness stage)
├─ Slower conversion (5-10% in first visit)
├─ Message: Education → Trust → Benefit → Buy
├─ ROAS: 2-4:1 (good, but slower burn)

STRATEGY:
├─ Google: Urgency & problem-solving
├─ Meta: Education & lifestyle inspiration
└─ Both: Multi-touch approach wins
```
